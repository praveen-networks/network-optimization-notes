## 📶 Wi-Fi Optimization Basics

- Avoid DFS channels in high-density environments  
- Maintain proper channel separation  
- Prefer 5 GHz band for better performance  
- Monitor client load per access point  

---

## ⚠️ Common Issues

### Frequent Disconnections
- Check RF interference  
- Reduce channel overlap  
- Validate firmware stability  

### Slow Speeds
- Check client distribution  
- Verify backhaul capacity  
- Analyze signal strength
